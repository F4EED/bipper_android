# Asset partagé Android ↔ client web

Fichiers canoniques pour l’icône **envoi / gestion alerte** (disque rouge + gyrophare bleu) :

| Fichier | Usage |
|:--------|:------|
| `core/resources/.../drawable/ic_bipper_alert_siren.xml` | Android / Desktop (Compose) |
| `shared-assets/bipper_alert_siren.svg` | Copier vers `client_web_MT_bipper/apps/web/public/images/bipper_alert_siren.svg` |

Après modification du visuel, mettre à jour les deux emplacements pour garder la parité.
